Running a Node

Installation

Install golang

wget https://go.dev/dl/go1.21.6.linux-amd64.tar.gz

# extract
tar -C /usr/local -xzf go1.21.6.linux-amd64.tar.gz

# add to path
export PATH=$PATH:/usr/local/go/bin

# check 
go version

Install Docker

# Add Docker's official GPG key:
sudo apt-get update
sudo apt-get install ca-certificates curl gnupg
sudo install -m 0755 -d /etc/apt/keyrings
curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo gpg --dearmor -o /etc/apt/keyrings/docker.gpg
sudo chmod a+r /etc/apt/keyrings/docker.gpg

# Add the repository to Apt sources:
echo \
  "deb [arch=$(dpkg --print-architecture) signed-by=/etc/apt/keyrings/docker.gpg] https://download.docker.com/linux/ubuntu \
  $(. /etc/os-release && echo "$VERSION_CODENAME") stable" | \
  sudo tee /etc/apt/sources.list.d/docker.list > /dev/null
sudo apt-get update

# Install latest docker version
sudo apt-get install docker-ce docker-ce-cli containerd.io docker-buildx-plugin docker-compose-plugin

# verify successful installation with 
sudo docker run hello-world

Install Eigen

# install Eigen CLI using binary
curl -sSfL https://raw.githubusercontent.com/layr-labs/eigenlayer-cli/master/scripts/install.sh | sh -s

# add to PATH
export PATH=$PATH:~/bin

Setup

Create Keys

# create Keys 
eigenlayer operator keys create --key-type ecdsa operator 
eigenlayer operator keys create --key-type bls operator

Create config

eigenlayer operator config create

Use https://ethereum-goerli.publicnode.com for RPC

Edit metadata.json and upload to publicly available URL.

Then edit operator.yaml to point to this metadata.json file.

Add EL_Slasher_Address: 0xD11d60b669Ecf7bE10329726043B3ac07B380C22 Add BLS Compendium Address: 0xc81d3963087Fe09316cd1E032457989C7aC91b19

Register

eigenlayer operator register operator.yaml

need testnet eth

Stake ETH

Stake ETH on Goerli

  • Stader Staked ETH

  • Token Contract ETHx

  • ANKR Staked ETH

  • Token Contract

Deposit on EigenLayer here

DA

Clone Repo and setup

git clone https://github.com/Layr-Labs/eigenda-operator-setup.git
cd eigenda-operator-setup
cp .env.example .env

Edit .env

  • ECDSA and BLS private keys
  • node IP
  • rpc url
  • Key Passwords

Make required local folders

mkdir -p $HOME/.eigenlayer/eigenda/logs
mkdir -p $HOME/.eigenlayer/eigenda/db

Register as AVS for EigenDA

# register
sudo ./run.sh opt-in

# run eigenDA 
sudo docker compose up -d

# validate
sudo docker ps
